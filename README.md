# Bus-Management-System
A simple Java-based application for managing bus routes, schedules, passenger records, and ticket bookings. Built to demonstrate object-oriented programming principles, data management, and efficient transportation system operations. 

## System Modules

### Login System
- Provides secure access to the Bus Management System.
- Authenticates users before allowing access to management features.

### Bus Management
- Add new buses to the system.
- Remove existing buses.
- Assign passengers to buses.
- Assign routes to buses.
- View bus occupancy and available seats.

### Driver Management
- Add new drivers.
- Remove drivers from the system.
- Assign drivers to buses.
- View driver details and assignments.

### Passenger Management
- Register new passengers.
- Remove passenger records.
- View booked tickets.
- Access passenger travel history.

### Ticketing System
- Book tickets for passengers.
- Check seat availability.
- Calculate ticket fares.
- Generate passenger tickets.
- View fare collection reports.

### Route Management
- Add new routes.
- Remove routes.
- View route details and schedules.
- Track passenger booking history for specific routes.

### Exit System
- Allows users to safely exit the application at any time.
